# OmniBudget: The All-Ages Expense Tracker

OmniBudget is a "lifecycle" financial application that adapts to different age groups with unique user interfaces and features tailored to each stage of life.

## Features

### Multi-Profile System
- **Explorer (Ages 6-12)**: Gamified financial learning with colorful interface
- **Pacer (Ages 13-22)**: Social features and budget management for young adults
- **Builder (Ages 23-60)**: Comprehensive financial tools for wealth management
- **Guardian (Ages 60+)**: High-contrast, accessible interface for seniors

### Core Features
- Income and expense tracking
- Budget management with different approaches per age group
- Savings goals with progress tracking
- Gamification elements for younger users
- UPI integration for Indian users
- Local SQLite database for secure data storage

## Technology Stack

- **Backend**: Python with Flask framework
- **Database**: SQLite for local data storage
- **Frontend**: HTML5, CSS3 with theming, JavaScript
- **Templating**: Jinja2 for dynamic content rendering

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Initialize the database
5. Run the application
6. Open your browser and navigate to `http://localhost:5000`

## Usage

1. Register for an account and select your age profile
2. Add transactions to track income and expenses
3. Set budgets to manage your spending
4. Create savings goals and track your progress
5. Switch between profiles to experience different interfaces
